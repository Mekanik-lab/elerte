# Magazynek IT

Prosta aplikacja webowa do zarządzania magazynem IT, napisana w **PHP**, **MySQL**, **JavaScript** i **Bootstrap**.

Projekt umożliwia:
- zarządzanie produktami w magazynie,
- rejestrowanie wydań,
- prowadzenie inwentaryzacji,
- przeglądanie historii operacji,
- zarządzanie kontami 

## Funkcje

### Użytkownik
- logowanie do systemu,
- przegląd magazynu,
- dodawanie produktów,
- edycja i usuwanie produktów,
- wydawanie produktów,
- tworzenie i uzupełnianie inwentaryzacji,
- zatwierdzanie własnej inwentaryzacji,
- zmiana loginu,
- zmiana hasła,
- dezaktywacja konta.

### Administrator
- wszystkie uprawnienia zwykłego użytkownika,
- podgląd użytkowników,
- podgląd historii operacji,
- rejestracja nowych użytkowników.

### Dane logowania administratora
- **login:** `admin`
- **hasło:** `admin123`

## Technologie

- **PHP 8+**
- **MySQL / MariaDB**
- **JavaScript**
- **Bootstrap 5**
- **XAMPP**
